# 담당자별 계획서

각 담당자는 자신의 폴더에 작업 계획서를 작성합니다.

| 폴더 | 담당자 | 영역 |
|------|--------|------|
| [classifier/](classifier-A/) | 담당자A | Classifier 서비스 |
| [guidance-B/](guidance-B/) | 담당자B | Guidance 엔진 로직 |
| [guidance-C/](guidance-C/) | 담당자C | Guidance Knowledge Base |

## 계획서 작성 규칙

1. 파일명: `PLAN-<작업내용>.md` (예: `PLAN-noise-reduction.md`)
2. 각 폴더의 `TEMPLATE.md`를 복사하여 사용
3. 작업 완료 후 계획서 하단의 상태를 `완료`로 변경
